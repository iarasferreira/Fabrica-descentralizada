# Fabrica-descentralizada
FÁBRICA DESCENTRALIZADA - SIMULAÇÃO MULTI-AGENTE (SPADE)

Objetivo:
Simulação de uma fábrica descentralizada (SPADE) onde agentes (Máquinas, Robôs, Suppliers, Crews) coordenam tarefas essenciais usando o protocolo CNP (Contract Net Protocol).

Visualização:
Em tempo real, através do Matplotlib, mostrando o layout dinâmico dos agentes em colunas verticais e o status (Livre/Ocupado/Falha).

Dependências (Python 3.8+):
- spade
- numpy
- matplotlib

Como Executar:
1. Instalar dependências: `pip install spade numpy matplotlib`
2. Iniciar: `python main.py`
3. O programa solicitará o número de agentes de cada tipo.
4. A simulação será iniciada numa janela Matplotlib.
